# 🕵️‍♂️ 환경요인이 범죄율에 미치는 영향 분석
본 프로젝트는 서울시의 환경적 요인(유흥주점 수, CCTV 수, 인구 밀도, 지역 상권 등)이  
범죄율에 어떤 영향을 미치는지 데이터 기반으로 분석한 연구 프로젝트입니다.

이 README에서는 사용된 데이터, 분석 과정, 모델링, 시각화, 결론을 정리하여 누구나 쉽게 따라 할 수 있도록 작성하였습니다.

---

## 📌 프로젝트 개요
최근 도시 환경 요소들이 범죄 발생 패턴에 영향을 준다는 연구가 많아지고 있습니다.  
본 프로젝트는 다음 질문에 답하고자 합니다.

> **“환경 요인은 범죄율에 어떤 방식으로 영향을 미치는가?”**  
> **“특정 요인이 범죄 증가와 높은 상관관계를 보이는가?”**

이를 위해 서울시 25개 구를 중심으로 여러 환경 데이터를 수집하여  
이들 요인과 범죄율 사이의 관계를 분석하였습니다.

---

## 📊 사용된 데이터
아래 데이터는 2018~2024년 또는 최신 연도를 기준으로 하며, 모두 공공데이터 포털 및 서울시 열린데이터광장에서 수집했습니다.

| 항목 | 설명 |
|------|-------|
| 🔹 **범죄 데이터** | 5대 범죄(절도, 폭행, 강도, 살인, 성범죄) 발생 건수 |
| 🔹 **유흥주점 데이터** | 지역별 유흥주점 수(인허가 기반) |
| 🔹 **인구 데이터** | 지역별 총 인구 및 인구 밀도 |
| 🔹 **CCTV 설치 현황** | 범죄예방용 CCTV 설치 대수  |
| 🔹 **비상벨 설치 현황** | 범죄예방용 비상벨 설치 대수  |
